# ALBA-RL-for-VOS
Implementation of ALBA: RL for VOS. Accepted in BMVC 2020.
This uses a simple RL based approach for unsupervised video object segmentation. Please refer https://arxiv.org/pdf/2005.13039.pdf for the paper.
